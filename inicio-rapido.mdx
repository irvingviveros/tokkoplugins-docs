# Inicio Rápido — 5 Minutos

¿Quieres empezar a sincronizar propiedades de inmediato? Sigue estos pasos sencillos para tener Tokko Plugins funcionando en tu sitio WordPress.

<Steps>

<Step title="Instala y activa el plugin">

**Opción A: Desde el panel de WordPress**
1. Ve a **Plugins** → **Añadir nuevo**
2. Busca "Tokko Plugins" en el directorio oficial
3. Haz clic en **Instalar ahora**
4. Haz clic en **Activar**

**Opción B: Descarga manual desde Freemius**
1. Ve a [Freemius Dashboard](https://freemius.com)
2. Descarga el archivo ZIP de Tokko Plugins
3. En **Plugins** → **Añadir nuevo**, selecciona **Subir plugin**
4. Carga el archivo ZIP y activa

<Tip>
Si no ves el plugin en el directorio oficial, probablemente uses el plan Starter. Accede a través de Freemius o contacta soporte.
</Tip>

</Step>

<Step title="Verifica que Houzez esté activo">

El plugin funciona **solo con el tema Houzez**. Compruébalo:

1. Ve a **Apariencia** → **Temas**
2. Busca **Houzez** en la lista
3. Si no está activo, haz clic en **Activar**

<Warning>
Si no tienes instalado el tema Houzez, el plugin mostrará una advertencia. Instala y activa Houzez primero, luego vuelve a Tokko Plugins.
</Warning>

</Step>

<Step title="Accede a la configuración del plugin">

1. En el menú de WordPress, ve a **Tokko Plugins**
2. Selecciona **Configuración** (o **Settings**)
3. Verás la sección **Conexión con Tokko Broker**

</Step>

<Step title="Obtén tu API Key de Tokko Broker">

Tu **API Key** es la credencial que conecta WordPress con tu CRM. Para obtenerla necesitas tener rol **Administrador** en tu cuenta de Tokko Broker.

1. Inicia sesión en [tokkobroker.com](https://www.tokkobroker.com/) con tu usuario Administrador.
2. Entra a **Mi Empresa** → [tokkobroker.com/company](https://www.tokkobroker.com/company/).
3. Haz clic en la pestaña **Permisos** → [tokkobroker.com/company/permissions](https://www.tokkobroker.com/company/permissions).
4. En el apartado **Api Key**, selecciona el código y cópialo.

<Info>
**Guía oficial de Tokko Broker:** [¿Cómo saber cuál es la API KEY de mi cuenta?](https://dudas.tokkobroker.com/es/articles/432304-como-saber-cual-es-la-api-key-de-mi-cuenta) — incluye el video explicativo paso a paso.
</Info>

<Warning>
Si no ves la sección **Permisos** o el campo **Api Key**, tu usuario no tiene rol de Administrador. Pide la clave a alguien con ese rol o que te asignen los permisos correspondientes.
</Warning>

</Step>

<Step title="Ingresa el API Key en Tokko Plugins">

1. En **Tokko Plugins → Configuration**, localiza la tarjeta **API Connection**.
2. Pega la clave completa que copiaste desde Mi Empresa → Permisos.
3. Haz clic en **Test Connection** para validar la conexión en vivo.
4. El badge de la tarjeta pasará a **Connected** (verde) si todo es correcto.

<Tip>
Si ves un error, verifica que:
- La clave esté completa, sin espacios al inicio o final.
- Tu servidor WordPress tenga cURL habilitado.
- La cuenta de Tokko Broker esté activa y no haya sido regenerada recientemente (al regenerarla, la clave anterior deja de funcionar inmediatamente).
</Tip>

</Step>

<Step title="Ejecuta la primera sincronización">

Ahora es el momento de traer tus propiedades a WordPress:

1. En **Tokko Plugins** → **Sincronización** (o **Sync**)
2. Haz clic en el botón **Sincronizar Ahora** (o **Sync Now**)
3. Verás un indicador de progreso: "Descargando propiedades..."

<Warning>
**Primera sincronización:** Dependiendo de cuántas propiedades tengas, puede tardar:
- Menos de 100 propiedades: **1-2 minutos**
- 100-500 propiedades: **5-10 minutos**
- Más de 500 propiedades: **15-30 minutos** (o más)

**No cierres el navegador** ni recargues la página. Puedes ver el progreso en el dashboard.
</Warning>

</Step>

<Step title="Verifica que las propiedades se sincronizaron">

Cuando termine la sincronización:

1. Ve a **WordPress** → **Propiedades** (o **Properties**)
2. Deberías ver una lista con tus propiedades de Tokko Broker
3. Abre una propiedad para verificar que contenga:
   - ✓ Título y descripción
   - ✓ Imágenes/galería
   - ✓ Precio y detalles
   - ✓ Ubicación en mapa

Si ves tus propiedades, **¡felicitaciones!** Tokko Plugins está funcionando correctamente.

<Tip>
Si no ves propiedades:
- Asegúrate de que Tokko Broker tenga propiedades activas
- Verifica que el API Key sea válido (paso 5)
- Mira la sección **Troubleshooting** si necesitas ayuda
</Tip>

</Step>

</Steps>
